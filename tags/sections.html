Body

The body is a semantic element that contains all the content of the document.

Article

Article on Articles

An article is a semantic element which represents a part of the article that is self contained and can be syndicated. There can be many articles in a document.

Section

A section is a semantic element that represents a categorical grouping of information.

Navigation

A nav section is a semantic element that show which links are navigational links.

Aside

An aside is a semantic element that holds information that may not be directly related to the container it is in.

H1-6

These simply represent headers of a section/grouping.

They are located in the <aside> tag.

Header

The header is a semantic element that contains introductory information about the container it is in.

Footer

The footer is a semantic element that contains the “footer” of its container... okay.

Address

The address is a semantic element that contains the contact information of the writer of the content.
