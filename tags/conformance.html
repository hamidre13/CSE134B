Conformance Page

Given that Html5 has too many elements and attributes, we decided to highlight only the differences here. Following are the Highlighted differences:

  • a

    • download attribute does not work on internet explorer

  • attr

    • I.E. does not style the element

    • Firefox shows a dotted lunder underneath

    • Chrome does nothing, unless inside dfn tag

  • bdi

    • I.E does not support the element

  • ol

    • reverse attribute not working on I.E.

  • data

    • Not supported by Chrome

    • Not supported by I.E.

  • q

    • I.E. does not add quotes around the body

  • ruby

    • Not supported by I.E.

    • Not supported by Firefox

  • video

    • webm players appear different depending on browser

  • embed

    • chrome does not have flash-player by default (on linux)

  • object

    • typemustmatch tag only works for Firefox

  • keygen

    • I.E. does not support keygen

    • Firefox only says “Medium Grade” or “High Grade”

    • Chrome says “(1024) Medium Grade” or “(2048) High Grade”

  • input

    • color not supported on I.E.

    • date not supported on Firefox, I.E.

    • datetime not supported on Firefox, I.E., and Chrome

    • month not supported on Firefox, I.E.

    • number behaves differently

      • On Chrome, up-down arrows appear when focused

      • On Firefox, up-down arrows always present

    • search

      • Chrome has (x) that clears searchbar

      • Firefox does not

    • week is not supported

  • meter

    • not supported on I.E.

  • output

    • not supported on I.E.

  • select

    • looks between Chrome and Firefox different

  •  

  • caption

    • Firefox captions not visible for audio only files

  • details

    • not supported in I.E. or Firefox
